Secure Authentication

Role-Based Access

Sistem role Admin dan User dengan permission yang berbeda. Admin memiliki full access ke semua fitur.

CRUD Universities

Create, Read, Update, Delete data universitas lengkap dengan upload gambar (validasi jpg/png, max 2MB).
